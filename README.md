# How to run the app

## Run dbs-server

1. Configure your db in config.py
2. Activate your virtual environment
3.
```
cd ./dbs-server/
pip3 install -r requirements.txt
python init_db.py
python app.py
```


## Run admin-server

```
cd ./admin-server
npm i
npm start

```

# Run client

```
cd ./client
npm i
npm start
```
